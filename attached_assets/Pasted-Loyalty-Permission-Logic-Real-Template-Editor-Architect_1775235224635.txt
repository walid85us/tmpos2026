  - DO NOT show “Upgrade Plan” to store users

Implement or fix two distinct loyalty permissions:
- Edit Customer Loyalty

- Manage Loyalty Settings

- If user has Edit Customer Loyalty only:
  - they can edit loyalty tier / points in customer profile

- If user has Manage Loyalty Settings only:
  - they can access loyalty settings/tier management
  - customer profile loyalty edit controls should remain read-only unless Edit Customer Loyalty is also granted

  - they can do both
- If user has neither:
  - no Upgrade Plan message
  - no settings-management access
  - customer loyalty fields are read-only/disabled
