==================================================
PRIMARY GOAL
==================================================

Ensure escalated support cases are included in Command Center Needs Attention.

A support case should appear in Needs Attention when:
- case.escalated === true
- and case status is not Resolved / Closed
- or equivalent active-status logic used by the Support Tools data model

The item should show:
- priority: high or critical, depending on severity
- type: Support Case / Escalated Case
- tenant if available
- case title/subject
- escalation reason if available
- case id
- action link to /owner/support-tools?caseId=<id>

Investigate why escalated support cases do not appear in Command Center Needs Attention.

- CommandCenterPage.tsx Needs Attention derivation
