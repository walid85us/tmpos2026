PRIMARY GOAL
==================================================

Review and improve Phase 1.1.1 across all major surfaces so it feels modern, dynamic, interactive, and operationally mature.

The System Owner should immediately understand:

- What needs attention?
- What is getting worse?
- What is overdue?
- Which tenant is most at risk?
- Which audit/security event should be reviewed?
- Which support case needs escalation/acknowledgement?
- Which widget or pulse item can take me to the right place?
- What is the next operational action?

Use deterministic, transparent, rule-based operational intelligence.

Do not implement AI in this pass.

==================================================

Review the current Mission Control header against mature SaaS operations-console patterns.

Improve as needed.

Required:
- stronger top-level visual hierarchy
