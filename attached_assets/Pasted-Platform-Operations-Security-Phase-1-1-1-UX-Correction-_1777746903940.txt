==================================================

Upgrade the Command Center header so it feels like a modern operations “Mission Control” surface.

Required:

  - Healthy
  - Watch
  - At Risk
  - Critical

PART B — FOCUS MODE MUST VISIBLY CHANGE THE UI
==================================================

Focus Mode must have visible effect.

Modes:
- Normal
- Watch
- Incident Review

Required behavior:

Normal:
- show all widgets/items

Watch:
- prioritize or filter to:
  - watch / at-risk tenants
  - warning / critical audit events
  - at-risk / overdue SLA cases
  - escalated cases
  - failed/pending domains
- visually show “Watch Mode Active”

Incident Review:
- show only or strongly prioritize:
  - critical support cases
  - overdue SLA
  - escalated cases
  - critical/high-risk audit events
  - failed domains
- visually show “Incident Review Active”
- hide low/noise items where practical

Required:
- Needs Attention queue must visibly change between modes if data supports it.
- Widget counts/distributions should update or display filtered context.
- If no items match a mode, show a clear empty state:
  “No incident-level items for the selected time range.”

Do not make focus mode decorative.

==================================================

Make operational pulse items interactive.
